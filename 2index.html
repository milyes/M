Commandes IA Disponibles

Voici la liste des commandes d'intelligence artificielle que vous pouvez utiliser :

Commande : IA_WEBVIEW_LAUNCH

Lance une application WebView avec des paramètres spécifiques.

Paramètres:

  • --apk <nom_apk>: Spécifie le fichier APK à lancer.
  • --mode <online|offline>: Définit le mode de fonctionnement (en ligne ou hors ligne).
  • --theme <light|dark>: Choisit le thème de l'interface (clair ou sombre).

Exemple: IA_WEBVIEW_LAUNCH --apk interface.apk --mode offline --theme dark

Commande : AUTRE_COMMANDE_IA

Description de l'autre commande IA.

Paramètres:

  • --param1 <valeur>: Description du paramètre 1.
  • --param2 <valeur>: Description du paramètre 2.

Exemple: AUTRE_COMMANDE_IA --param1 valeur1 --param2 valeur2
