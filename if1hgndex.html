IA Commands Dashboard

Explore all available artificial intelligence commands.

Generate Image

Creates a visual representation based on a textual description. Perfect for bringing ideas to life.

/generate-image [description]

Translate Text

Translates input text from one language to another, supporting multiple languages.

/translate [source_lang] [target_lang] [text]

Summarize Document

Provides a concise summary of a given document or long text passage.

/summarize [document_url_or_text]

Code Generation

Generates code snippets or full functions based on a functional description or problem statement.

/code-gen [language] [description]

Sentiment Analysis

Analyzes text to determine the emotional tone (positive, negative, neutral).

/sentiment [text_to_analyze]

Text Completion

Continues a given text prompt, generating coherent and contextually relevant text.

/complete-text [prompt]
